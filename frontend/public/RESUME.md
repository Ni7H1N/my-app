# /public assets

## resume.pdf
The current `resume.pdf` is a **placeholder**. To publish your real resume:

1. Export your resume as `resume.pdf` (keep the filename exactly).
2. Drop it into `/app/frontend/public/resume.pdf`, overwriting the placeholder.
3. Refresh — the "Resume" button in the navbar and the "Download resume" CTA in the hero will serve the new file.

No code changes needed.
